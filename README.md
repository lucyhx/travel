# travel

## 攻略1

一、定位及花费

此次在三亚共待了10天，因为是随心游，所以基本是度假兼休闲，玩一天休整一天，2人共计花费15000大洋。在这里特别强调的是，三亚是适合各种消费水平的地方，旅游是当地的支柱产业，因此相关配套设施极其发达，住宿从几十块的青年旅社到几千的海边五星酒店，公交网络覆盖所有景点，小吃从几块钱的特色粉到成百上千的海鲜大餐，应有尽有，尽可以选择最适合自己的消费水准。

二、三亚景点

各类景点官方简介请找度娘，这里只谈本人的几点印象，优缺点一起说。

1.看海。三亚主打看海，海的成色、沙滩等都比内陆的好，本人在旅游时碰到一个试潜水的专业潜水员，就是专门负责到新开的海域查看是否适合潜水、并写评论的那种，她到过国内很多地方进行专业潜水，据她所述，三亚的海质是国内最好的，因此海景不容错过。适合看海的主要有三个湾，亚龙湾的海最好最美最清澈，海岸线长度在三个湾中处于中等，但离市区远，周边酒店最贵且吃不便宜，购物、吃饭、出行都不方便，可以在旁边住一到两个晚上。穷游的人建议即使不住那边也要去看看，可从市内搭公交去，通过公共海滩进入亚龙湾沙滩，待一个下午也很惬意；大东海的海清澈程度其次，离市区很近，但海岸线不长，虽然海滩上俄罗斯美女很多，但沙滩上人像下饺子似的，沙滩上的躺椅还要钱，不爽；三亚湾最长，但海水质量也最差，比起亚龙湾来差了不止一个档次，优点也多，由于海岸线长，沿海都是看海公寓，价格便宜还能180度看海，再加上就在市区交通方便，能吃到正宗特色小吃。另外还有很多海湾，本人大致去了海棠湾、清水湾、富力湾，也一并简单说一下。海棠湾风浪很大，冲击力强，适合冲浪，看海感觉一般，如果去免税店的话可以从免税店后面抄小路走5分钟就到了，想冲浪的话建议住一个晚上，旁边星级酒店很多；清水湾离南湾猴岛很近，水质一般，浪不大，看着没什么感觉，不如出名的三个湾；富力湾离市内较远，周边是度假小区，浪很大适合冲浪，但周边配套不全，似乎是专为每年度假一个月左右的人群所建。

2.岛屿：有蜈支洲岛、分界洲岛、西岛等，没开发但已经小有名气的还有加井岛等。蜈支洲岛和分界洲岛都是五A级景区。首先要说明的是，岛离市区都比较远，主打是玩潜水、摩托艇等项目，如果只是为了看海请不要去岛上，毕竟看不了几分钟就会腻，而周边的配套设施又相对较少。几个岛中景色最美的是蜈支洲岛，岛很小，纯走路的话一个小时就能转完，但景色确实漂亮，海水最清澈，据说被私人旅游集团承包了，外面的旅游团队都进不去，因此相应的价格最贵，门票网上150基本没有商量余地，各类潜水、摩托艇、拖伞等项目也不便宜，潜水带拍照近700块，摩托艇5分钟300元，建议只玩1个，如果真想多玩几个的话，有一个一口价全包，大概1300左右，可以一次玩到爽。分界洲岛景色其次，也是主打潜水、浮潜等项目，西岛离市区最近，相应的景色也最一般。据说还有个太阳湾也是潜水宝地，可以在港口潜水，适合潜水经验较丰富的人，本人没去，等待他人验证。所有岛屿因为都是在海上，风浪较大，去之前要先注意天气，即使天气好一般也是下午4、5点钟就发最后一班船了，因此要注意游玩时间。

3.景区：景点主要有3类，一类是主打佛教文化，南山寺（南海观音佛像）、大小洞天、天涯海角等，都在一条线上，建议花一天时间去其中两个地方即可，这三个地方应该是最早开发的旅游卖点，信佛的朋友可以去看看，108米的南海观音佛像不容错过（其实飞机上就可以看到）。这类景点建议坐市内公交直达即可。第二类是森林公园类，如呀诺达、槟榔谷、亚龙湾森林公园，主打热带氧吧，主要是在热带雨林中观赏热带植物，欣赏黎苗民族原住民的生活环境。呀诺达森林公园、槟榔谷是开发较早较出名的景点，亚龙湾森林公园因《非诚勿扰2》拍摄而出名，3个地方基本大同小异，建议选择一个地方即可，因为你看完之后不会留下多深刻的印象，各种各样的树而已。游玩时一定要跟团游或者购买讲解器，毕竟没有导游介绍的话植物基本分不清楚，可玩性大减。这类景点建议报当地一日游。第三类是特色风景区类，如南湾猴岛、五指山、鹿回头公园等，此类景区基本名气不大。南湾猴岛离市区较远，车程约一个小时，岛上猴子众多，有一些动物马戏表演，适合老人和孩子，但可玩性不高，基本2个小时就能逛完。鹿回头公园坐落在市内，离大东海很近，有时间的话建议抽出2-3小时逛逛，可以在山上俯瞰大东海和三亚湾，景色很漂亮，旁边还能看到野猴子，门票大概30元左右（含观光车），在市区内逛的时候可以作为一个备份参考地点。

三、三亚美食

首先要说的是，三亚的美食千万不要参考大众点评、美团等网站，都是刷出来的，虽然完全违背一般理论，很奇葩但却是事实。

1.海鲜。总的感觉，三亚的海鲜水很深！尽管靠海，可价钱一点都不便宜，再加上是旅游城市，海鲜已经产业化，价钱基本跟内陆同种类同品质的价钱差不多，有的还贵些。市场上天然海鲜和家养海鲜鱼龙混杂，很难分辨，旅游时有的当地人就说真正海鲜基本不卖给外地人，汗!吃海鲜大致有三个地方，第一市场、春园海鲜市场、下岗职工海鲜市场，都是类似大排档的地方，就餐环境很是一般。这些市场吃都是自个去买海鲜或者商家小妹带你去，商铺主要赚加工费。①第一市场是最早开发的市场，里面的海鲜市场极其火爆，人多、吆喝声多，卖海鲜跟卖白菜似的。因此挑选的时候一定要注意，坚持自己的挑选标准，不要被外部环境所动摇。紧邻第一市场有很多加工店，可以先在这条街上逛逛，看看各家的营业状况再决定去哪，毕竟开了几十年，要一直红火的店也不多，这样的店做的都是口碑和味道了。网上评价较好的是小胡子海鲜、四川毛哥海鲜、梅姐海鲜等，个人去了小胡子，味道还可以。这里要补充一下，第一市场旁边的夜市还不错，属于老牌成熟夜市，氛围挺好，适合女生边走边逛边看，小东西都挺便宜，有点特色的东西如果看中可以大胆砍价，成交价大致是开口价的一半左右。②春园美食广场。本地人评价不高，最早的海鲜市场，后来转移到第一市场，基本没落了，想去的也可以试试。在春园找加工店不同于第一市场，要找新开的又有人气的，因为新开的店会用心做口碑，相对来讲会好一些。③下岗职工海鲜市场。就是个大排档市场，店名都是几排几号，环境很是一般，不过好几家店味道不错。原本想推荐一家店的，在她们家吃了三四次，无论是海鲜、味道还是服务态度都很不错，但是很不幸，本人有一次晚上去吃，正好碰上快收摊，不小心地看了店铺的小姑娘与海鲜店的老板正在一起算账（分赃），人与人最基本的信任瞬间被打击得体无完肤，只能说，三亚的海鲜水真的很深。④常见海鲜的吃法。所有的海鲜都可以清蒸，这里说的是当地较出名又好吃的做法。开边蒜蓉：波士顿龙虾、对虾；椒盐：濑尿虾、基围虾；清蒸：老虎斑（石斑鱼的一种）、海胆蒸蛋、鲍鱼仔；蒜蓉粉丝蒸：扇贝、象拔蚌、生蚝；香辣炒：和乐蟹、基围虾、芒果螺、圣子；一只波士顿龙虾可以两种吃法，头部熬粥，尾部清蒸，不过加工费也是算两份的。⑤几点注意事项：街上的拉客小妹都不要理，不然会一直纠缠；挑海鲜的时候不要被现场的氛围左右，一定要慢慢挑，建议不要店里的小妹陪同去，自己多问几家，提前在网上搜一下怎么分别海鲜新不新鲜，货比三家基本没问题；称海鲜的时候要谨防无良商家掺水，最好提前带一瓶500ml或者250ml的农夫山泉，称重时可以把袋子抠破一个洞让水流出来；盯住商贩的手，以防掉包；蟹要捏一下，避免壳大无肉，虾避免活虾和死虾一块称，鱼要看眼睛，贝壳要不粘连的、好看的才有活性。

2.特色小吃。①海南四大美食，和乐蟹、文昌鸡、东山羊、加积鸭，很可惜，本人都没吃到正宗的，味道没觉得哪个很好吃值得推荐。②本地蔬菜中，四角豆真的不错，很好吃，建议到一般的饭店吃，如果要在吃海鲜的地方吃的话，一定要自己去菜市场买然后让饭店只收加工费，不然价格会是2倍。③本地的抱罗粉、海南粉、清补凉、椰子冻、鸡屎藤，我全都吃到了正宗的，超好吃，现在想想都流口水，椰子饭也挺不错，这些在外贸路都能吃到正宗的。外贸路是一条不大的路，延伸的还有外贸二路、外贸三路等，据说是本地的小吃一条街，甜蜜蜜的清补凉，小区内不知名店铺的海南粉，外贸路和友谊路交接路口的椰子冻，外贸二路不知名早点店的鸡屎藤，味道真的是太好了。外贸二路还能吃到后安粉，也是当地的名小吃，不过因为本人抗拒猪肝，就没有尝试。这些小吃都很便宜，抱罗粉一般是10块，海南粉8块、10块都有，清补凉一般要20或25，鸡屎藤3块钱满满一大碗，椰子冻要贵一些，基本35左右，有次贪便宜在大东海附近买了个25的，一点都不正宗。要说明的是，抱罗粉大清早是买不到的，新鲜的抱罗8点左右才能送到店家的小摊里，所以不要太早去吃早点。三亚其实是个夜城市，街边的小店有的都是上午10点才开门，营业到晚上2点，所以掌握他们的作息有利于更好地安排旅游休闲。

四、三亚购物

1.水果。海南盛产芒果、火龙果、椰子。椰子有青椰、黄椰、光头椰之分，黄椰最划算，一般9块左右一个，喝最鲜嫩的汁水，还可以吃椰子肉。水果不要到超市买，又贵又不好，本人到当地最大的旺豪超市购物，水果是最不花算的。个人吃的话可以到普通的水果市场购买，价钱和内地差不多，但已经是那边的良心价了，批发的话建议做18路去一个叫做红沙什么的地方（忘了，可以问一下当地人，都知道的），是一个很大的水果批发市场，三亚所有的水果源头都在这里，批发快递回家或者快递送人都是个很不错的选择。

2.其他特产。本地的老牌子有南国和春光，好吃点的有椰子糖（春光牌椰子糖很有小时候的味道，非常正宗）、椰子粉、牛轧糖等，我买的不多，看个人爱好。通过大量比价，本人发现，从网上买比在当地买要划算，可以在旅游时对每样感兴趣的特产都买一点试吃，选择好特产和品牌后，回到自己居住的城市用淘宝买都可以买到正宗的，而且价钱还便宜些，又省去了旅途携带重物的劳累，何乐而不为！

五、出行准备

1.机票。机票早定，作者提前了10多天观察机票，头天看好的隔了二天（真的就隔了两天）价格就涨了300块，吓得我赶紧订了，后来发现果然是明智的，离出行越近机票价格越贵。机票在官网上订最便宜，可以通过携程、去哪儿网、驴妈妈等第三方软件筛选适合自己的航班，再到相应航空公司的官网去订票，有时候官网的特价机票能便宜到你想哭，春秋航空不定时还有1元抢特价机票。需要说明的是，比价时要注意，携程等第三方软件显示的价格只是机票价格，付款时会把燃油附加费等其他费用都加上，而官网通常显示的是最终成交价格，因此刚看到官网时都会认为携程等软件的便宜，其实是假象。建议机票尽量选择下午到达三亚的，因为飞机降落前在建立起落线时，会绕108米高的南海观音转一圈，在飞机上看还是非常壮观的，作者认为飞机上看了地面上都不用再去这个景点了。

2.酒店。

①时间。酒店不同于机票，提前几天订就可以了（如果看中的酒店很抢手就另当别论）。三亚的酒店不同于内地，基本是按旅游的旺季淡季分的，全年哪个月哪天的价格基本固定，比如说春节期间五星酒店一般是4000左右、家庭酒店大概600左右，11月底到2月份是旺季，五星是1500、家庭公寓是300，基本不变，早订晚订区别不大。只要房源充足都可以晚订，留够时间观察自己中意的酒店是否有活动。比如我定的亚龙湾红树林，当时受机票事件影响，提前了近20天，后来发现酒店又出了一些其他活动，同样的酒店比我定的还便宜一百块，郁闷坏了。

②方式。通过携程等第三方软件预定，相对来讲比价方便，价格也比官网便宜，如果通过booking等平台预定的要注意，给的价格往往不含服务费（五星级酒店），要当心。通过大量查找酒店发现，三亚的星级酒店基本都是不可退的，不可退的，不可退的，重要的事情说三遍。因此下单付款前一定要看好，确定自己的行程和假期再付款。

③住哪。纯属个人推荐。

亚龙湾：住两天即可，毕竟除了海啥都没有，酒店贵，吃饭贵还不方便，附件的百花谷说是商业区，但里面连必胜客这类连锁店都是宰人不商量，完全没必要长住。亚龙湾的红树林、天域二期是网上公认性价比较高的酒店，红树林属于老牌五星级酒店，处于亚龙湾海岸线正中心，视野比较开阔，不过服务一般，里面的泰餐厅号称三亚最正宗的泰餐厅，价格也不便宜，2人大约400块，不过味道确实不错，即使不住红树林也建议去他家的泰餐尝尝；天域二期（说的是二期，一期完全不咋地）主打亲子乐园，就在红树林边上，占地面积大，绿化非常好，适合儿童的设施多，不过本人没有小孩所以没住，有小孩的可以试试。作者沿亚龙湾海岸线走了一圈，海岸线旁边都是五星级酒店，总体感觉差不多，各家星级酒店都有自己的特色，没有哪家非常好或非常不好，美高梅的泳池位置就非常不错。同时发现，这里的五星级酒店说是私人海滩，其实并没有私人化，整个是连起来的，你可以从这个酒店的沙滩走到另一个酒店的沙滩，未住店的人也可以从公共下海通道进入。

大东海：最早开发的区域，消费适中，有优质海景房，也不会太贵，海水质量、酒店价格都处于三个湾的中间地位，单论看海的话性价比最高。因为开发早，商业配套比较多，相应的逛街、吃饭的地方也比较多，但个人认为都商业化了，反而吃不到正宗的本地味道。要提醒一下，大东海附近有个礁石区，非常美，此地靠近万豪酒店，沿大东海往万豪方向走，钻过一个桥墩，再走过一片砂砾地就到了，大概走个10分钟。游人基本不去那，但是风景真的很好，据说是三亚当地拍婚纱照必去的地方，还可以在那里抓寄居蟹等，其他湾都没有看到活的生物，唯独这里。

三亚湾：离市区最近的海湾，有“椰梦长廊”景点，房价最便宜，但海水质量也是最差的。三亚湾很长，只有2-3公里离市区较近，订酒店要注意，尽量选择在此区域内，不然性价比就不如大东海了。唯一可以安慰的是海岸线比较长，住哪都可以看海，花最少的钱就能看到180度海景，而且海景比大东海、亚龙湾都要开阔。（配图）

关于海景房、园景房的选择：直接看海当然要比在房间内看海爽，请理智消费。如果很想要在房间看海的感觉，建议亚龙湾可以选择高层海景房（海水最好），大东海、三亚湾可以选择90度或180度无敌海景房。

入住时间：星级酒店一般下午才能入住，如果上午到的话，请提前联系酒店看能否提前入住。

关于家庭公寓：受前期海南炒房影响，很多小区在建设时就设计了适合各类人群的半海景、全海景房，从一室一厅到三室二厅应有尽有，很多商家购买不同小区的房间后，设立酒店对外住宿，就成了家庭公寓，也叫家庭旅馆。房间内除基本设施外还自带冰箱、洗衣机、烟机灶具等，非常适合一家人旅游时短期住宿，价格相对酒店要便宜得多。此类家庭公寓遍布三亚湾、大东海，基本都在成熟小区内。

个人认为亚龙湾适合住1-2日，纯待在海边，看海；三亚湾多住几日，吃饭较为方便，可以报当地的旅游团游玩景点、潜水等，早出晚归待在房间内的时间不多，可以选择房价较低的家庭公寓（也能满足海景要求哦）；大东海定位与三亚湾差不多，但房价整体贵一些，购物、吃饭很方便但味道确实一般，建议住1-2个晚上即可。

温泉别墅：本人极力推荐，一定要住一晚上。首先纠正误区，温泉并不是只有冬季泡，天气热时照样可以泡。其次，临近海棠湾的南田附近有很多温泉度假区，离免税店公交只要15分钟，入店可以送一张免税店95折的卡，很实用，我在免税店花了近4000块，靠95折的折扣都省了200。第三，600左右就能订购一个别墅，真的是别墅哦，非常划算。我住的是南田温泉别墅度假酒店（不是打广告），是那个专业试潜的妹子推荐的，当时还特意叮嘱我住靠近餐厅的房间，非常感谢她。酒店对每个入住客人都送温泉票，温泉区有鱼疗池，是那种全身都可以沉到水里被鱼疗的那种池子，其他池子就和一般温泉差不多了。别墅上下两层，分成两户，每户都自带一个温泉池，可以自己在房间里泡温泉，私密性非常好，感觉很爽。入住房间都送一顿自助早餐，比起其他五星级酒店，自助餐真一般，不过作者觉得整体性价比很高，住的别墅、屋内有喷泉、还可以鱼疗，已经不错了，愿意尝试温泉的可以去试一下，酒店应该是个五星级酒店（也可能四星、不确定），从大东海有个直发的班车，其他方式去的话就不知道了。

3.交通。真心话，三亚不建议自驾游，也不建议当地租车，车就是个累赘，理由有四：一是市区较小，外地车多，吃海鲜的地方更是几条旧街道，开车停车都很麻烦，不方便；二是市区公交系统方便，到各个景点都有公交，双层巴士、普通专线都能满足一般旅客要求，举个例子，去凤凰机场的公交车停的站台与的士是一模一样的，下公交车即进候机厅，开车还麻烦些，机场都能如此，其他的自己想吧；三是三亚适合休闲度假，有时候在海边躺个一两天，车空放着很浪费；四是不适合公交的景点，报当地团最划算，包接送，自驾去还贴油钱，而且旅游团在来回车程、门票上基本是不赚钱的。当时碰到一姐们报了个36块的团去分界洲岛（去分界洲岛来回车程至少2个小时，门票大概是180块），她只花了36块！！！当时真是羡慕嫉妒恨。可想而知三亚当地的旅游行业竞争激烈和门票的水分有多大了。不过开车也有一个好处：三亚的沿海公路比较长，开车在海边兜风还不错，自个取舍吧。

4.随身用品。①防晒霜。紫外线很猛烈，同时太阳伞、墨镜也别忘了，太阳帽可以到当地买，二三十的质量就很好，用完即扔。②洗漱用品，酒店的都不太卫生，至少要带2匹毛巾，三亚沿海，换衣服的场合比较多，身边有条干毛巾会让女生觉得你是贴心小暖男。③背包。建议带一个双肩旅行背包和一个随身小包，随身小包用来放钱包、手机等贵重物品，双肩包则是能玩爽的重要保证，草帽、墨镜、泳衣、替换衣物等日常用品都可以放在双肩包内。④地图。手机里提前下载好一份三亚地图，方便导航和GPS。⑤现金。三亚毕竟不是乡下，日常几百块现金足够，除了海鲜要付现外，其他基本都用不了大额现金，可以根据个人要吃的海鲜来确定身上的现金额。
